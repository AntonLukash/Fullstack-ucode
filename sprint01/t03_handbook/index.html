HTML

Introduction to HTML

HTML, which stands for HyperText Markup Language, is the standard markup language for creating web pages and web applications.

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My First HTML Page</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first HTML page.</p> </body> </html>

HTML Tags

HTML tags are the building blocks of an HTML document. They define the structure and content of different elements on a web page.

<h1>Heading 1</h1> <p>This is a paragraph.</p> <img src="image.jpg" alt="Image">

HTML Attributes

HTML attributes provide additional information about HTML elements. They are placed within the start tag of an element and are used to modify the element's behavior or provide metadata.

<img src="image.jpg" alt="Image"> <a href="https://example.com">Link</a>

HTML Forms

HTML forms are used to collect user input. They allow users to enter data that can be submitted to a server for processing.

CSS Selectors

CSS selectors are patterns used to select and style HTML elements. They can be used to target elements based on their tag name, class, id, or other attributes.

h1 { color: blue; } .container { width: 100%; }

CSS Box Model

The CSS box model describes the structure of every HTML element on a web page. It consists of content, padding, border, and margin.

.box { width: 200px; height: 200px; padding: 20px; border: 1px solid black; margin: 10px; }

PROTOCOLS

HTTP/HTTPS

HTTP (Hypertext Transfer Protocol) and HTTPS (HTTP Secure) are protocols used for transmitting data over the internet. HTTP is used for unencrypted communication, while HTTPS encrypts the data to provide a secure connection.

HTTP methods include GET, POST, PUT, DELETE, and more. Response status codes range from 1xx to 5xx, indicating different types of responses such as informational, success, redirection, client error, and server error.

DNS

The Domain Name System (DNS) is a hierarchical and decentralized naming system for computers, services, or other resources connected to the internet or a private network. It translates domain names, such as example.com, into IP addresses, allowing browsers to locate resources on the internet.

DNS records include various types such as A, AAAA, CNAME, MX, TXT, and more, each serving different purposes like mapping domain names to IP addresses, defining mail servers, and verifying domain ownership.

IP

IP (Internet Protocol) is the principal communications protocol in the Internet protocol suite for relaying datagrams across network boundaries. It provides an identification and location system for computers on networks and routes traffic across the internet.

To determine the IP address of a domain name, you can use various methods such as using the command line tool "nslookup" or "ping" on Windows, or "dig" or "host" on Unix-like systems. For example, to find the IP address of google.com, you can use the command "nslookup google.com" or "ping google.com".

FTP

FTP (File Transfer Protocol) is a standard network protocol used for the transfer of computer files between a client and server on a computer network. It is commonly used to transfer website files from a local computer to a web server for hosting.

FTP operates on two ports: 21 for sending commands and 20 for sending data. It supports various commands such as LIST, RETR, STOR, and more for navigating directories, downloading, and uploading files.
